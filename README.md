# Loyalty Backend

Backend sederhana untuk program loyalty dengan OTP dan form pendaftaran.
Dibangun pakai:

- Node.js (Express)
- PostgreSQL
- JWT (`jsonwebtoken`)
- Zod (validasi input)
- Pino & pino-http (logging)
- dotenv (konfigurasi environment)

---

## 1. Requirements

- Node.js 22.x
- PostgreSQL 16.x (atau kompatibel)
- Git (opsional, untuk clone & versioning)

---
